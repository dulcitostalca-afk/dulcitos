Cómo obtener tu Client ID de Google:

1. Entrá a console.cloud.google.com
2. Creá un proyecto nuevo → ponele "Dulcitos FUDO"
3. Buscá "Google Calendar API" y hacé clic en Habilitar
4. Andá a Credenciales → Crear credenciales → ID de cliente OAuth 2.0
5. Tipo de aplicación: Aplicación web
6. En "Orígenes autorizados" agregá: https://claude.ai
7. Copiá el Client ID que aparece y pegalo abajo
Lo obtenés en Google Cloud Console (es gratis y el proceso tarda ~5 minutos)
